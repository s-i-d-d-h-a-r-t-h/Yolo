# YoloV3:
You Only Look Once.

Detection of koala bears using yolo v3 algorithm.

Configuration:
- Batch size = 64 images
- Subdivisions = 16 images = No. of minibatches
- Size of a minibatch = 64/16 = 4
- Filters = (no. of classes + 5)*3 = (1+5)*3 = 18
